Overview

Clarkston's Computing and Enginnering Club has worked on a weather station to collect data from the Clarkston campus. The weather station is made from a rain gauge that sends data to an Arduino that has weather sensors to detect temperature, pressure, and humidity.


Parts of the Project


General Setup Arduino Setup Rain Tip Weather Data Sensor


The Arduino is in a container to prevent rain damage. Attached to it is a shield for an SD card and batteries.

The magnet changes position, records the time, and every tip is 100 mL of rain.

Sensor to calculate temperature, air pressure, and relative humidity.


How the Data Is Collected

The data from the sensor (including temperature, pressure, and humidity) and time of the rain tip are stored on the SD card. The data is stored in a text file on the card that is read by JavaScript and seperated line by line. After each line has been put into an array, each line is split into another array with temperature, air pressure, humidity, and time.

Calculations

To calculate we first converted the Unix Time to days by dividing that time by 86,400 and taking the remainder to find how many seconds had passed in that day and divided it by 3,600 to find the hour of the tip. The hour was now in decimal form, so we converted the decimal part of the hour to minutes.
